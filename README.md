<h1 align="center">🛍️ Laravel CRUD Product Management System</h1>

<p align="center">
A modern and responsive Product Management System built using <strong>Laravel</strong>, <strong>PHP</strong>, <strong>MySQL</strong>, <strong>Bootstrap</strong>, and <strong>MVC Architecture</strong>.
</p>

<p align="center">
This project demonstrates complete CRUD (Create, Read, Update, Delete) functionality along with image upload management, status handling, database operations, routing, controllers, models, migrations, and Laravel best practices.
</p>

<hr>

<h2>📌 Project Overview</h2>

<p>
The Product Management System is designed to help administrators manage product information efficiently.
</p>

<p>The application allows users to:</p>

<ul>
<li>✅ Create Products</li>
<li>✅ Upload Product Images</li>
<li>✅ Edit Existing Products</li>
<li>✅ Delete Products</li>
<li>✅ View Product Listings</li>
<li>✅ Manage Product Status</li>
<li>✅ Store Data in MySQL Database</li>
<li>✅ Follow Laravel MVC Architecture</li>
</ul>

<p>
This project was developed to strengthen practical knowledge of Laravel fundamentals and backend development concepts commonly used in real-world applications.

<h2>📦 Installation Guide</h2>

<h3>Clone Repository</h3>

<pre>
git clone https://github.com/your-username/LARAVEL-CRUD.git
</pre>

<h3>Navigate to Project Directory</h3>

<pre>
cd LARAVEL-CRUD
</pre>

<h3>Install Composer Dependencies</h3>

<pre>
composer install
</pre>

<h3>Create Environment File</h3>

<pre>
cp .env.example .env
</pre>

<h3>Generate Application Key</h3>

<pre>
php artisan key:generate
</pre>

<h3>Configure Database Credentials</h3>

<p>
Update the <strong>.env</strong> file with your MySQL database details.
</p>

<h3>Run Database Migrations</h3>

<pre>
php artisan migrate
</pre>

<h3>Start Development Server</h3>

<pre>
php artisan serve
</pre>
